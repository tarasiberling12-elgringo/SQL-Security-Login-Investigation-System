## Project Title

# SQL Security Login Investigation System

⸻

# Overview

This project analyzes login attempt data using SQL to detect suspicious behavior, classify user risk levels, and identify potential brute-force attacks.

⸻

# Features

* Failed login detection
* User-level aggregation
* Risk classification using CASE WHEN
* Suspicious user filtering (HAVING)
* Ranked security analysis
____

# Example Output 

admin | 19 | CRITICAL
root  | 14 | CRITICAL
john  | 1  | LOW
____

# Skills Used

* SQL filtering (WHERE)
* Aggregation (COUNT, GROUP BY)
* Conditional logic (CASE WHEN)
* Analytical sorting (ORDER BY)
* Security mindset thinking
